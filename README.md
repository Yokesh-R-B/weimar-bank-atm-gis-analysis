# GIS Analysis of Banks and ATMs in Weimar

## Project Overview
This project analyzes the spatial distribution of banks and ATM locations in Weimar using GIS tools. The goal is to visualize the accessibility of financial services and identify geographic coverage across the city.

## Tools Used
* QGIS
* OpenStreetMap data
* GeoJSON / GeoPackage spatial formats

## Project Objectives
* Map banks and ATM locations in Weimar.
* Perform attribute and spatial queries.
* Analyze geographic accessibility of financial services.
* Generate a cartographic map for visualization.

## Data Sources
* OpenStreetMap banking infrastructure dataset.
* Public geographic datasets for Weimar.

## GIS Methods
The analysis includes:
* Spatial layer creation.
* Attribute filtering for banks and ATM locations.
* Spatial visualization and map styling.
* Export of a final cartographic map.

## Output
Final map showing the spatial distribution of banks and ATMs in Weimar.
